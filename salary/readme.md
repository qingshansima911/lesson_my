## js 数据类型
    简单数据类型
        - number 不分长短整型，都是数值类型
        - string
        - boolean
        - null 空 定义了值为空
        - undefined 未定义 声明了变量但为赋值
            内存空间（undefined有地址，但地址长度不定） 弱类型语言
    复杂数据类型
    - object
        除了简单数据类型，都是对象
        array []可遍历的对象
        object {} 对象字面量
        function 可执行的对象
    -自学作业 typeof 可获得数据类型但不准确
